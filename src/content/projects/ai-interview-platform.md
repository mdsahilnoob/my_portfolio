**AI Interview Platform** is a comprehensive full-stack application where users can practice mock interviews with an AI interviewer. The system conducts real-time interviews, records responses, and generates detailed performance reports with feedback.

## Key Features

- **Real-time Interview Sessions**: Live Q&A powered by Socket.io
- **AI Integration**: Intelligent question generation and response evaluation
- **User Authentication**: Secure JWT-based auth system
- **Performance Reports**: Automated scoring with strengths/weaknesses analysis
- **Modern UI**: React with Vite for optimal development experience
- **Scalable Architecture**: Clean separation of concerns with services layer

## Tech Highlights

- **Backend**: Express.js with MongoDB, modular architecture with controllers/services
- **Frontend**: React (Vite) with context-based state management
- **Real-time**: Socket.io for live interview rooms
- **Database**: MongoDB for users, sessions, and reports
- **Authentication**: JWT middleware for secure routes
